# SpotifyMusic for Stand

SpotifyMusic can use wav files which you can import each favorite music, it's for Stand Lua API Documentation. Inspired by Lance to get Music Sounds.

Inspired X-Force features which we can upload own musics but Stand don't have these features so I decided to create and using some luas script to support.

Installation
--------------------

1. Copy 'SpotMusic.lua' to %appdata%\Stand\Lua Scripts
2. Copy 'lib' and 'store' folder into %appdata%\Stand\Lua Scripts
3. Done and enjoy your music.

NOTE: One of musics having different format files like .mp3, .wgg, .ogg, etc... are not compatible with Stand. So be advise, use .wav converter music to work.
